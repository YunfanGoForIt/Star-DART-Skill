# 自动分类规则

## 分类表

| 一级分类 | 二级分类 | 关键词 |
|----------|----------|--------|
| AI智能体框架 | 多智能体、单智能体、编程智能体 | agent, ai-agent, autonomous, framework |
| 深度研究 | 智能体研究、论文分析 | research, deep-research, scientific |
| 学术论文 | Arxiv、翻译、摘要 | paper, arxiv, academic |

| 生物医学 | - | biomedical, medical, biological |
| 其他 | - | 无法匹配上述分类 |

## 匹配规则

1. 优先匹配一级分类关键词
2. 其次匹配二级分类关键词
3. 都无法匹配 → 其他
