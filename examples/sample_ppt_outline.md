# Star-DART OPC 汇报 PPT 大纲

## 1. 封面

Star-DART OPC：办公小浣熊驱动的开源项目情报与知识资产工作台

## 2. 痛点

GitHub Star 越来越多，但真正需要时找不到、想不起、懒得重读，收藏无法变成能力。

## 3. 解决方案

使用默认 3 小时定时轮询发现新增 Star，再用办公小浣熊完成研究、档案、表格、看板、PPT 和周报。

## 4. 办公小浣熊能力地图

本地 Agent、Skill、专家团、docx/pdf/xlsx/pptx、data-dashboard、知识库、第三方数据、定时任务。
